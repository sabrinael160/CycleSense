# Proyecto Spring Boot - RegistroDolor

# Descripción
Proyecto de ejemplo con Spring Boot para gestionar registros de dolor.
Incluye CRUD completo (Crear, Leer, Actualizar, Borrar) y una interfaz web con Thymeleaf.

## URL de acceso a la web
Una vez iniciado el proyecto, acceder a la web en:

http://localhost:8080/registros

## Cómo ejecutar el proyecto
1. Abrir la carpeta del proyecto en VS Code o IntelliJ.
2. Ejecutar `SpringumaApplication.java` como aplicación Java.
3. Abrir un navegador y usar la URL indicada.

## Funcionalidades
- Listar registros
- Crear nuevo registro
- Editar registro existente
- Borrar registro
